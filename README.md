# Lista de Exercícios em PHP

## Descrição

Este repositório contém uma lista de exercícios desenvolvidos em **PHP puro**, com o objetivo de praticar o uso de funções, organização de código e boas práticas de programação.

Cada exercício foi desenvolvido em um arquivo separado (`ex01.php`, `ex02.php`, etc.), utilizando funções para tornar o código mais organizado, reutilizável e de fácil manutenção.

## Exercícios

### Exercício 01
Calcula o valor da expressão `(x² + y²) / (x + y)`, verificando se a divisão por zero pode ocorrer antes de realizar o cálculo.

### Exercício 02
Recebe um texto, exibe sua quantidade de caracteres e apresenta o texto invertido.

### Exercício 03
Recebe um CPF e aplica uma máscara, ocultando todos os dígitos, exceto os quatro últimos.

### Exercício 04
Gera uma senha aleatória com o tamanho informado, utilizando letras, números e caracteres especiais.

### Exercício 05
Realiza uma análise de um texto, informando:
- Quantidade de palavras;
- Quantidade de caracteres;
- Quantidade de vogais;
- Quantidade de consoantes.

### Exercício 06
Converte temperaturas entre Celsius (°C) e Fahrenheit (°F).

### Exercício 07
Calcula o desconto aplicado a uma compra conforme o valor total, exibindo o valor original, o desconto e o valor final.

### Exercício 08
Ordena uma lista de nomes em ordem alfabética.

### Exercício 09
Analisa um número, informando se ele é:
- Par ou ímpar;
- Primo ou não;
- Perfeito ou não.

### Exercício 10
Analisa um conjunto de notas, exibindo:
- Maior nota;
- Menor nota;
- Média;
- Situação final do aluno.

### Exercício 11
Realiza diferentes formatações em um texto:
- Converte para maiúsculas;
- Converte para minúsculas;
- Deixa apenas a primeira letra maiúscula;
- Conta a quantidade de caracteres.

### Exercício 12
Analisa uma lista de produtos, informando:
- Produto mais caro;
- Produto mais barato;
- Média dos preços;
- Pesquisa de um produto pelo nome.

### Exercício 13
Implementa a Cifra de César para criptografar e descriptografar mensagens utilizando um deslocamento de caracteres.

### Exercício 14
Calcula estatísticas de uma lista de números, incluindo:
- Soma;
- Média;
- Maior valor;
- Menor valor;
- Mediana;
- Quantidade de números pares;
- Quantidade de números ímpares.
